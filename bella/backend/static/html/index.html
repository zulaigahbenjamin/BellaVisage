Users Table

HTTP Methods Path Description
GET /users displays a list of users
GET /user/:id displays a specific user based on id
POST /user Creates a new user
PATCH/PUT /user/:id Updates an existing user or adds a user
DELETE /user/:id Deletes a user

Products Table

HTTP Methods Path Description
GET /products Displays a list of products
GET /product/:id Displays a single user by id
POST /product Adds a new product
PUT/PATCH /product/:id Updates a products details
DELETE /product/:id deletes a product from the database
